# Fetcher

This program fetches apple affiliate conversion data

## Usage

```
./fetcher -appKey=xxx -apiKey=xxx -from=2017-02-01T00:00:00 -to=2017-02-02T00:00:00 -go=4 -host=127.0.0.1:3306 -user=jason -pwd=jason -db=fenda -log_dir=/tmp
```
